INSTALLING TAILWIND CSS
-> npm init  		(have node.js installed)
-> npm install -D tailwindcss 
-> npm install vite
-> npx tailwindcss install
-> Go to tailwind.config.js and set the content to ["*"]
-> Install tailwind intellisense extension for vs code

-> npm run start : run the website runs on the localhost
